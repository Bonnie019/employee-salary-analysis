# employee-salary-analysis
This project focuses on analyzing employee salary data using MySQL and Python. The MySQL database was connected to VS Code for efficient query execution and integration with Python scripts. For data analysis and visualization, the following libraries were installed and utilized:

Pandas for data manipulation and cleaning

Seaborn for statistical data visualization

Matplotlib for creating detailed plots and charts

The workflow combines SQL queries for structured data extraction with Python-based visualization to uncover trends, patterns, and insights in employee salary distributions.
